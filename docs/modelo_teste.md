# Modelo de Teste — ChargeGrid Assistant

## Testes Funcionais

Pergunta: Qual o status dos carregadores?
Resposta esperada: mostrar carregadores ativos.

Pergunta: Existe risco de sobrecarga?
Resposta esperada: informar consumo e limite contratado.

Pergunta: Qual carregador está em falha?
Resposta esperada: identificar erro.

Pergunta: Qual o consumo atual?
Resposta esperada: mostrar potência total.

Pergunta: Quantos carregamentos estão ativos?
Resposta esperada: informar sessões simultâneas.

---

## Testes de Variação de Linguagem

Pergunta: Os carregadores estão funcionando?
Resposta esperada: mostrar status dos carregadores.

Pergunta: Estamos perto do limite de energia?
Resposta esperada: informar risco de sobrecarga.

---

## Testes Fora de Escopo

Pergunta: Qual a previsão do tempo?
Resposta esperada: informar que a pergunta não pertence ao sistema.

Pergunta: Quem fundou a empresa?
Resposta esperada: fora do escopo operacional.
