## Modelo de Teste do Chatbot

1. Qual o status dos carregadores?
Resposta esperada: mostrar carregadores ativos.

2. Existe risco de sobrecarga?
Resposta esperada: informar consumo e limite.

3. Qual carregador está em falha?
Resposta esperada: identificar erro.

4. Qual o consumo atual?
Resposta esperada: mostrar potência total.

5. Quantos carregamentos estão ativos?
Resposta esperada: informar sessões simultâneas.
